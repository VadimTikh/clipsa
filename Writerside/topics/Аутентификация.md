# Аутентификация

API использует токен-аутентификацию для всех запросов. Токен необходимо передавать как параметр запроса.

## Параметры аутентификации

| Параметр | Тип | Обязательный | Описание |
|----------|-----|-------------|----------|
| `token` | String | Да | Токен доступа к API |

## Процесс аутентификации

1. Токен должен быть добавлен в качестве параметра запроса к каждому API-запросу: `?token=ваш_токен`
2. Токен проверяется на соответствие значению, установленному в переменной окружения `process.env.TOKEN`
3. При несовпадении токена API вернет ошибку 401 (Unauthorized)

## Пример запроса с аутентификацией

```
GET /api/baf/products?token=ваш_токен
```

## Ответ при ошибке аутентификации

```json
{
  "message": "Invalid token"
}
```

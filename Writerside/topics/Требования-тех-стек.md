# Требования (тех-стек)

---

## Язык и окружение

- **Язык программирования:** TypeScript `^5.6.3`
- **Среда выполнения:** Node.js `^22.x`

---

## Сервер и маршрутизация

- **Фреймворк:** Express
- **HTTP клиент:** Axios 
- **Прокси:** https-proxy-agent

---

## База данных

- **MongoDB:** через официальный драйвер `mongodb`

---

## Планировщик задач

- **Планировщик:** `node-cron`

---

## Конфигурация и окружение

- **Переменные окружения:** `dotenv`

---

## Инструменты разработки

### Зависимости

- `axios`
- `dotenv`
- `eslint`
- `express`
- `https-proxy-agent`
- `mongodb`
- `node-cron`
- `nodemon`
- `ts-node` — запуск TypeScript кода без сборки
- `typescript` — компилятор TypeScript
- `gts` — стиль кода от Google

---

## Скрипты (`npm scripts`)

- `dev` — запуск проекта в режиме разработки - файл index.ts (`nodemon`)
- `dev_test` — запуск проекта в режиме разработки - файл test.ts (`nodemon-test.json`)
- `build` — сборка TypeScript в JavaScript
- `start` — запуск собранного кода (`build/src/index.js`)
- `fix` — автоматическое исправление стиля

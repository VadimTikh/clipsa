# Роуты

Документация по доступным маршрутам API.

## Группа: BAF

### GET /api/baf/products

Получение списка ВСЕХ спарсенных товаров для BAF.

#### Запрос
- Метод: `GET`
- URL: `/api/baf/products`
- Требует аутентификацию: Да

#### Ответ

**Успешный ответ (200 OK)**

```json
{
  "data": [
    {
      "sku": "string",
      "supplier_name": "string",
      "name": "string",
      "id": "string",
      "cost_price": "number",
      "availability": "boolean"
    }
  ]
}
```

**Ошибка (500 Internal Server Error)**

```json
{
  "error": "Описание ошибки"
}
```

## Группа: Content

### GET /api/content/products

Получение информации только о связанных закупкой товарах, с актуальными ценами и наличием для Клипсы.

#### Описание

**Особенности:**
1. Это товары, которые были обработаны закупкой и одобрены для добавление на сайт
2. Себестоимость может быть 0, если товар не найден ни у поставщика, ни в CRM
3. Цена продажи может быть 0, если отсутствует себестоимость
4. Наличие `false`, если отсутствует себестоимость
5. Правила наценки берутся из правил Техномикса (хардкодены в БД временно). Если правило не найдено, используется дефолтная наценка 25%

#### Запрос
- Метод: `GET`
- URL: `/api/content/products`
- Требует аутентификацию: Да

#### Ответ

**Успешный ответ (200 OK)**

```json
{
  "data": [
    {
      "id": "string",
      "sku": "string",
      "title": "string",
      "cost_price": "number",
      "clipsa": {
        "sell_price": "number",
        "old_price": "number",
        "availability": "boolean",
        "hidden": "boolean"
      },
      "current_suppliers": [
        {
          "supplier_name": "string",
          "supplier_sku": "string",
          "supplier_id": "string"
        }
      ]
    }
  ]
}
```

**Ошибка (500 Internal Server Error)**

```json
{
  "error": "Описание ошибки"
}
```

### POST /api/content/update_clipsa_dop_nacenka

Обновление доп наценки для товара.

#### Запрос
- Метод: `POST`
- URL: `/api/content/update_clipsa_dop_nacenka`
- Требует аутентификацию: Да
- Content-Type: `application/json`

#### Тело запроса

```json
{
  "sku": "string",
  "dop_nac": "number"
}
```

| Параметр | Тип | Обязательный | Описание |
|----------|-----|-------------|----------|
| `sku` | String | Да | Артикул товара |
| `dop_nac` | Number | Да | Значение дополнительной наценки |

#### Ответ

**Успешный ответ (200 OK)**

```json
{
  "result": {
    "sku": "string",
    "price": "number"
  }
}
```

**Ошибка валидации данных (400 Bad Request)**

```json
{
  "error": "required body type: sku: string, dop_nac: number"
}
```

**Ошибка сервера (500 Internal Server Error)**

```json
{
  "error": "Описание ошибки"
}
```

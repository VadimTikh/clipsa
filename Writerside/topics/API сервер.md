# API сервер

Проект имеет API сервер, который позволяет обмениваться информацией с другими нашими сервисами.

✅ `GET /api/baf/products` — Получение всего списка спарсенных товаров для BAF  
✅ `GET /api/content/products` — Получение списка связанных (одобренных) закупкой товаров с ценами и наличием
✅ `POST /api/content/update_clipsa_dop_nacenka` — Обновление доп наценки для товара

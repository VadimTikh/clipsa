# Логи крон задач

Логи крон задач из этого проекта доступны в нашем веб приложении:
https://t-tech.pp.ua/dashboard/reports/cron-logs

## Названия логов

Логи этого проекта имеют названия, которые начинаются со слова **Клипса**:

- `Клипса - сохранить в БД актуальную информацию о товарах поставщиков`
- `Клипса - обновить и добавить товары в СРМ со склада`

## Просмотр логов

Для просмотра логов:
1. Перейдите на страницу https://t-tech.pp.ua/dashboard/reports/cron-logs
2. Используйте поиск по слову "Клипса" в списке "Задачи"
3. Выберите интересующую задачу для просмотра подробной информации
